Save files in the same location as StdDraw.java

For the Queens.java file, cd into file location and type

>>javac Queens.java StdDraw.java

to compile. Followed with

>>java Queens (any number from 4-16)

For QueensSym.java, it's the same as the first except the command to compile
will be

>>javac QueensSym.java StdDraw.java
>>java QueensSym (number)
